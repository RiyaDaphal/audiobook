# 📚 Python PDF Audiobook Converter

A lightweight Python script that converts PDF text into spoken audio (MP3). This tool is designed to work even in "headless" cloud environments like GitHub Codespaces.

## 🚀 Features
- Extracts text from any standard PDF.
- Converts text to speech using `pyttsx3`.
- Saves output as an `.mp3` file for offline listening.

## 🛠️ Installation

Since this project uses external libraries, you must install them first:

```bash
